#폴더별 설명

## Build
    
    웹 페이지를 build 하는 폴더여서 거의 변경 X

## source
    웹페이지의 페이지를 설정하는 폴더 추가할 페이지는 source 폴더에 md, rst 파일로 추가

    source 에서 코드 변경 -> 변경 사항을 html 코드로 저장시키기 (.\make.bat html)
    
    src의 새로 생긴 코드를 rst 파일로 자동 변경
    (sphinx-apidoc -f -o docs/source/ src)
    
    (pycharm 터미널 기준)
